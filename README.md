Simple local Ai agent learned in youtube

Video : https://www.youtube.com/watch?v=E4l91XKQSgw&t=184s

Model: Llma 3.2
Embedder: mxbai-embed-large

the csv dataset is about pizza reviews and the model answer questions mostly about pizza

to run clone this repo, 
download Ollama and the models above,
and run: python main.py
